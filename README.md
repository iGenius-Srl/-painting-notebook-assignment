## Painting editor

### Specs
- The painting is a **16x16** pixel
- Create and **save** painting (in browser memory)
- **Edit** saved paints
- Can **tag** paints
- **Search** paints by tags/names
- Preset of **10 colors**
- export/import **json**
- **undo/redo**
